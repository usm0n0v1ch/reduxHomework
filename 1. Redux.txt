1. Redux
Store это центральное хранилище состояния приложения. Передает данные компонентам через Provider
Action/Reducer — Action описывает, что делать type, payload, Reducer обновляет состояние на основе Action
Преимущества — Упрощает управление состоянием в сложных приложениях, избавляет от проп-дриллинга
2. React Router DOM

Позволяет создавать маршруты и навигацию между страницами.
Используется через BrowserRouter, Routes и Route.
Передача данных между страницами через state в Link или URL-параметры.
3. Redux + React Router DOM

Redux управляет общими данными для всех страниц.
Передача данных через useSelector для компонентов внутри маршрутов.
Аутентификация — проверка состояния пользователя в Redux перед рендером маршрута (ProtectedRoute).
